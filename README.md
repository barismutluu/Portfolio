# Barış Mutlu Portfolio

Single-page portfolio built with React, TypeScript, Vite and Tailwind CSS.
It highlights software development, backend API, frontend and software test
automation projects from the public GitHub profile.

## Tech Stack

- React 18
- TypeScript
- Vite
- Tailwind CSS
- EmailJS
- Formik and Yup
- React Vertical Timeline

## Scripts

```bash
npm run dev
npm run lint
npm run build
npm run preview
```

## Project Structure

```bash
src/
  Assets/
  Components/
  Sections/
  App.tsx
  main.tsx
```

## Contact Form Environment

Copy `.env.example` to `.env` and define EmailJS values with the `VITE_`
prefix:

```bash
VITE_EMAILJS_SERVICE_ID=
VITE_EMAILJS_TEMPLATE_ID=
VITE_EMAILJS_PUBLIC_KEY=
```
