# Barış Mutlu Portfolio

Single-page portfolio built with React, TypeScript, Vite and Tailwind CSS.
It highlights software test automation, backend API and frontend projects from
the public GitHub profile.

## Tech Stack

- React 18
- TypeScript
- Vite
- Tailwind CSS
- EmailJS
- Formik and Yup
- React Vertical Timeline
